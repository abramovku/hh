# Laravel Pint - Стиль кода

## Что это?
Laravel Pint - это инструмент для автоматического форматирования PHP кода в соответствии со стандартами Laravel.

## Использование

### Исправить все файлы
```bash
.\pint.bat
```

### Проверить без исправления
```bash
.\pint.bat --test
```

### Проверить только измененные файлы (Git)
```bash
.\pint.bat --dirty
```

## Настройки

Конфигурация в файле `pint.json`:
- Preset: Laravel
- Single quotes для строк
- Автосортировка импортов
- Удаление неиспользуемых импортов
- Пустая строка перед return

## CI/CD интеграция

Добавьте в ваш CI pipeline:
```yaml
- name: Run Pint
  run: ./vendor/bin/pint --test
```

## Pre-commit hook (опционально)

Создайте `.git/hooks/pre-commit`:
```bash
#!/bin/sh
./vendor/bin/pint --dirty
git add .
```

Сделайте его исполняемым:
```bash
chmod +x .git/hooks/pre-commit
```
